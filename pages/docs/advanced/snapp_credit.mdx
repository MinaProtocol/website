### Ubuntu 20.04

Download the Ubuntu 20.04 version [here](""). Once downloaded, you can install by issuing the following:

```
sudo add-apt-repository universe
sudo apt update
sudo apt install -y ~/Downloads/snapp-credit-check-ubuntu20.04.deb
```

### WSL2 

It is possible to get the app running WSL2 but you will need to set up X11 forwarding to see app running on your local machine. If you already have X11 forwarding set up on your Windows
machine, you can use the [Ubuntu 20.04 version]("") to get started.
```
sudo apt update
sudo apt install -y /path/to/snapp-credit-check-ubuntu20.04.deb
````
